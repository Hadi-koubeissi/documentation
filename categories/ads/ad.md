# $-ad

| Usage | Permission Requirement |
| :---: | :---: |
| `$-ad` | None |

{% hint style="warning" %}
#### Precautions:

* This command can only be used in a server
* This command will only work if the server it's used in has a registered ad
{% endhint %}
